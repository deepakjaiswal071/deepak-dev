# Product Plan: AI Test Case Generator

## Objective
A token-optimized web application that transforms user stories into precise, deterministic QA test cases using structured LLM outputs.

## Tech Stack
- **Backend:** Python (FastAPI), Uvicorn, Python-dotenv, Pydantic (v2).
- **Frontend:** React (Vite), Tailwind CSS, Lucide React (icons).
- **Data Format:** Enforced JSON matching `.claude/schema.json`.
